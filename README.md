# unnamed-tdg

Unnamed Tower Defense Game

## Setup inicial

1. Primero descargar el proyecto

    Por ssh: 
    ```
    git clone git@github.com:algiorgi/unnamed-tdg.git
    ```

    o por http:
    ```
    git clone https://github.com/algiorgi/unnamed-tdg.git
    ```

1. Instalar git-lfs

    ```
    git lfs install
    ```
